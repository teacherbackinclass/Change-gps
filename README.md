# Change GPS (Beta)

A navigation tool for people going through organizational change. 

## Features
- **Change Map:** Visualize where you are in "The Dip."
- **Tools & Scripts:** Role-specific language for Individuals and Leaders.
- **Regulate:** A box-breathing tool to lower stress.

## Tech Stack
- React + Vite
- Tailwind CSS
- Lucide React (Icons)

## Quick Start
1. `npm install`
2. `npm run dev`
